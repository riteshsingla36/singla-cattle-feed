# Claude Code Memory Index

This directory contains memory files that persist between sessions to help continue work without repetition.

## Memory Files

- **project_status.md** - Complete project overview, current status, issues, and next steps

## How to Use

When you reopen this project in Claude Code, I will automatically read these memory files to recall:
- What has been built
- Current issues and their fixes
- Configuration details (Firebase project ID, etc.)
- Next steps to take

You don't need to repeat anything - just say "continue from last time" or ask questions, and I'll have full context.
